#### 注意：
`在win下不好使`

#### 安装：
`npm install -g node-inspector`

#### 启动inspctore服务：
`node-inspector &`

#### debug模式运行：
`node --debug index.js`

#### 结合supervisor：
`supervisor --debug index.js`
